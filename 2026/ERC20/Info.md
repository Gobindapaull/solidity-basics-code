## Create backend
- Create Project Folder
- npm init -y
- Install Packages: npm i express dotenv ethers cors
- Create .env
- server.js
- routes/token.js
- controllers/tokenController.js
- node server.js
- Test API
- POST request: http://localhost:3000/api/token/create
- Get Token Info: http://localhost:3000/api/token/info
- Transfer Token: http://localhost:3000/api/token/transfer
- GET balance: http://localhost:3000/api/token/balance/0x7E1FdF03Eb3aC35BF0256694D7fBe6B6d7b3E0c8
- Burn token: http://localhost:3000/api/token/burn
